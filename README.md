# Retail Sales Analytics Dashboard

## Description
An interactive retail sales analytics project built with MySQL and Power BI to analyze KPIs, sales trends, category performance, regional insights, and product-level profitability.

## Tools Used
- MySQL
- SQL
- Power BI

## Dashboard KPIs
- Total Sales: 2.33M
- Total Profit: 292.30K
- Total Orders: 5111
- Profit Margin: 12.56%

## Key Visuals
- KPI Cards
- Monthly Sales Trend
- Sales Distribution by Category
- Sales by Region
- Sales vs Profit by Product
- Region Slicer

## Key Insights
- Sales show strong seasonality, with peak performance in Q4.
- West region contributes the highest sales, while South contributes the lowest.
- Technology leads category sales contribution.
- Overall profit margin is 12.56%, indicating profitable business performance.
- Product-level analysis shows that high sales do not always translate into high profit.

This project demonstrates how SQL and Power BI can be used together to transform transactional retail data into actionable business insights through interactive dashboards.
